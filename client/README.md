### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd client-react
```

2. Install dependencies

```bash
npm install
```

3. Setup environment variables

```bash
cp example.env .env
```

4. Configure `.env` file with your settings

```env
VITE_API_URL=http://localhost:8080
```

5. Start development server

```bash
npm run dev
```
